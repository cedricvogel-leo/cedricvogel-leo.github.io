Webseite auf Github

Projekt erstellen

Account erstellen

Um eine Webseite auf Github zu publizieren brauchen sie einen Account. Den können sie hier erstellen: github.com/signup Sie brauchen dazu eine gültige Email-Adresse.

Neues Projekt

Melden sie sich bei Github an und klicken sie auf "Create Repository"
Der Name den sie dem Repository geben, muss die Form username.github.io haben, wobei username dem Benutzernamen entspricht, den sie gewählt haben. Sie müssen die Option Public anwählen.

Daten Hochladen

Nachdem sie das Repository erstellt haben, befinden sie sich auf der Seite, wo sie ihre Webseite verwalten können. (Die Url lautet github.com/username/username.github.io Wobei sie username wiederum durch ihren Benutzernamen ersetzen müssen)
Klicken sie auf "uploading an existing file:"

Ziehen sie den Inhalt ihres Webseiten-Ordners mit der Maus auf die Fläche, welche erscheint.

Klicken sie unten auf der Seite auf "Commit Changes". Nach einigen Minuten sollte ihre Webseite auf username.github.io sichtbar sein.

Webseite bearbeiten

Online Editor

Sie können ihre Webseite online mit Visual Studio Code bearbeiten. Dazu müssen sie auf folgende URL gehen:
github.dev/username/username.github.io
username entspricht auch hier ihrem Benutzernamen.

Änderungen speichern

Im Gegensatz zu der lokalen vevsion von Visual Studio code gibt es keinen Button zum speichern einer Datei, welche sie verändert haben.
Stattdesen müssen sie auf das "Source Control" Symbol am linken Rand das Browsers klicken, in der Message-Box ihre Änderungen beschreiben und dann auf das Häckchen (Commit and Push) klicken.
Auch hier dauert es einige Minuten bis die Änderungen auf der Webseite sichtbar sind.

Bilder Hosten

Wenn sie eigene Bilder auf ihrer Webseite haben, welche gross sind, kann es sinnvoll sein, diese an einem anderen Ort zu Hosten und einzubinden. Da der Speicherplatz auf Github begrenzt ist. Ein Möglichkteit ist zum Beispiel Google drive. Hier finden sie eine Anleitung zum Hosten von Bildern auf Google Drive.

Domain Name

Wenn sie ihren eigenen Domeinnamen für ihre Webseite haben wollen, müssen sie eine Domain Namen mieten. Für CH-Domains bezahlen sie etwa 15 Franken im Jahr.
Eine Liste mit allen Firmen, welche diese Domains offiziell vermieten finden sie hier. Wenn sie die Seite auf Github hosten brauchen sie nur den Domainnamen kein Hosting. Viele Der Firmen haben Pakete für Hosting und Domainnamen. Ein solches brauchen sie nicht.
Wenn sie eine Domain registriert haben wird ihnen hier erklärt, wie sie Ihre Domain für Ihre Seite auf Github nutzen können.
